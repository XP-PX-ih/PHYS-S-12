Program the Microcontroller

For this assignment, I opted to gain a head start on my final project, or at least familiarity with it. I decided on developing the one handed keyboard/mouse. Since that project requires a good number of buttons, about 10, I wanted to figure out how to map those buttons to letters now rather than later. That way I could spend more time focusing on how to create the shell and circuits for the mouse/keyboard rather than the programming. I had my fears, but I wanted to see if I could get the majority of the chorded layout of the keyboard done beforehand, so I began coding eight buttons for the microcontroller. I referred to the left handed ARTSEY Layout when programming them.
